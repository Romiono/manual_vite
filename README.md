### Spa template

Шаблон для быстрой развертки spa приложения на базе **react + vite**.

По умолчанию имеет настроенные vite, ts, eslint и prettier, husky, а также структуру fsd.

#### Vite

Настроен для работы с react как в дев режиме так и для сборки. По у умолчанию велючены sourcemap, для более простого дебага.

Настроена работа с svg, как с ReactComponent, но без возможности забирать их из public, рекомендуется хранить их в `share/assets`.

#### Ts

Настроен базовый конфиг для работы с vite.

Скрипты:

- `npm run dev` - запуск приложения
- `npm run build` - запуск сборки с параметром `mode=production`
- `npm run build:dev` - запуск сборки с параметром `mode=development`
- `npm run lint` - запуск линтинга на базе eslint и prettier (будьте внимательны, т.к. скрипт нацелен на исправление ошибок, а не только на их находку)
- `npm run prepare` - скрипт для конфигурации Husky, необходимо запустить сразу после развертки приложения

Параметр mode в конфигурации используется для выбора директории, в которую будет выгружен бандл после сборки, вы можете добавить дополнительные параметры в vite.config.ts.

#### Eslint и Prettier

Применены правила по умолчанию для ts, react, а также включены следующие опции:

- обязательные ";"
- одинарные кавычки
- двойные кавычки в jsx
- висячие запятые в многострочных объектах
- обязательные фигурные скобки для логических блоков
- обязательные скобки для аргументов стрелочных функций
- пробелы в однострочнныз объектах `{ something }`
- сортировка атрибутов в jsx

Также применен плагин для сортировки импортов **simple-import-sort**.

#### Husky

По умолчанию настроен pre-commit хук, который прогоняет линтинг перед коммитом.

#### Общее

Настроены алиса от src.
